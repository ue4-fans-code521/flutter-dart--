## 前端开发中常见的编程范式

### 概念:

```shell
命令式编程：
	命令式编程非常好理解，就是一步步给计算机命令，告诉它我们想做什么事情声明式编程：
声明式编程:
	通常是描述目标的性质，你应该是什么样的，依赖娜些状态，并且当依赖的状态发生改变时，我们通过某些方式通知目标作出相应
```



> * **命令式编程**
>   * 早先的原生的JavaScript
>   * jquery
>   * ......都是命令式编程开发.
> * **声明式编程**
>   * vue框架
>   * react框架
>   * angular框架
>   *  小程序
>   * flutter框架
>   * ...... 都是属于声明式编程.
